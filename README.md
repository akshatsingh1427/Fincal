## 📌 About
FinCal is an interactive, educational financial calculator built for everyday investors. It covers SIP, Lump Sum, Goal Planning & Fund Comparison with live charts, real-time calculations, and concept explainers — making mutual fund literacy simple, visual & engaging.

---

## ✨ Features

| Module | Description |
|--------|-------------|
| 🚀 SIP Calculator | Monthly investment growth with animated bar charts |
| 💎 Lump Sum | One-time investment compounding over time |
| 🎯 Goal Planner | Reverse-calculate SIP/lump sum needed for your goal |
| ⚖️ Fund Compare | Risk vs reward table + radar chart across fund types |
| 📚 Learn | 6 key concepts with formulas (NAV, CAGR, Compounding) |
| 🗺️ Rupee Journey | Visual story of how your money grows + cost of delay |

---

## 🛠️ Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Chart.js (bar, line, doughnut, radar charts)
- No frameworks, no install — runs in any browser
